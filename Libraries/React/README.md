# Learn to "React"
React.js の学習や情報収集のまとめ

- [React](https://facebook.github.io/react/)
- [Getting Started | React](https://facebook.github.io/react/docs/getting-started.html)

__ビルドツール__

```sh
$ npm install -g react-tools

# jsxコマンドが使える
$ jsx --watch src/ build/
# src/ 以下のファイルをコンパイルして build/ に配置
```
