# メディエータパターン

メディエータとは振る舞いのデザインパターンであり, 
システムのさまざまな部品が相互にコミュニケーションできるように
統一インターフェイスを公開するものです.

```js
let mediator = (function() {

  // ブロードキャストやリスンされるトピックのための記憶領域
  let topics = {};

  let subscribe = function(topic, func) {
    if (!topics[topic]) {
      topics[topic] = [];
    }
    
    topics[topic].push({
      context: this,
      callback: func
    });

    return this;
  };


})();
```
