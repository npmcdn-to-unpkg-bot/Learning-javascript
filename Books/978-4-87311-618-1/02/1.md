# 2.1 コンストラクタパターン

```js
function Car(model, year, miles) {
  this.model = model;
  this.year = year;
  this.miles = miles;
}

Car.prototype.toString = () => {
  return `${this.model} has done ${this.miles} miles.`
};

// test
var civic = new Car('Honda Civic', 2009, 200000);
console.log( civic.toString() );
```
